

                                    
------REST API SERVER DEMO BUILD. ▬▬▬█
                                                
All THE ROUTES LOGIC IS LOCATED INSIDE THE CONTROLLERS FOLDER.

START SERVER IN POWERSHELL, terminal command "npm run dev".

To access the server main endpoint "/" open browser and type "http://localhost:3050/".

-Access-endpoint "users" type in path "/users".
-Access-endpoint "id" type in path "/:id".

- when you start the server the users endpoint will only contain an empty array[]
- to test CRUD and add data to the mock database use POSTMAN.
